# synnergy_lab
Для того чтобы бот работал так как задумывался нужно:
## 1
Создать виртуальное окружение (при необходимости)

## 2
Установить все нужные библиотеки(pip install -r requrments)

## 3
Добавить в папку sckripts нужные скрипты (В них обязательно должна быть функция main, бот запускает именно её!)

## 4
Можно добавить прям в код команду (файл main.py там всё написано). Либо можно запустить бота и ввести команду на добавление скрипта в планировщик (планировщик сбрасывается при перезапуске бота).

## 5
Запусти бота и можешь запускать планировщик. Тут используется обычный sheduler (минуты, часы, дни, дни недели, месяц). 

## 6
Напиши команду /help.
Не забудь добавить и заполнить файл ENV

Приятного пользования.
